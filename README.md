## Introduction

This project allows users to play the classic Rock Paper Scissors game against a computer opponent. The user selects their move (rock, paper, or scissors) and the computer randomly selects its move. The game then determines the winner based on the rules of Rock Paper Scissors.

## Features

- Simple and user-friendly interface
- Real-time score updates
- Random computer moves
- Playable directly in the browser

## Technologies Used

- HTML
- CSS
- JavaScript

## Installation

To get started with the Rock Paper Scissor Game, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/sanjana2505006/Rock-Paper-Scissor.git
   ```
2. Open the `index.html` file in your web browser to start playing.

## Usage

1. Open the `index.html` file in your web browser.
2. Click on the rock, paper, or scissors icon to make your move.
3. The computer will randomly select its move.
4. The game will display the result and update the score.
